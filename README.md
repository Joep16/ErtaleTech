# ErtaleTech Inventory & Lead Management System

A web-based management system built using PHP and MySQL for handling leads and inventory.

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Features
- User authentication
- Admin dashboard
- Role-based access (Admin / User)
- Add, edit, delete records (CRUD)
- Lead management
- Responsive UI

## How to Run Locally
1. Install XAMPP
2. Start Apache & MySQL
3. Import `db/ertaletech.sql` using phpMyAdmin
4. Move the project folder into `htdocs`
5. Visit: `http://localhost/ErtaleTech`
